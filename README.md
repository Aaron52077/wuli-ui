# wuli-ui
- 基于小程序的ui
* 会持续维护和整理更新新的小功能,方便开发者提升效率哦~

## 参与开发
- `Aaron52077`
- 期待您的加入~ ,欢迎大家踊跃提交Issues。

## 功能分类
- 参考`zanUI`、`wux`特性整理一套比较基础应用功能
* 后期会新增和持续优化插件类功能

## 项目引用
- 项目结构中可以按照自身项目需求添加需要的`components`
    * 在`components`中找到你需要的伪组件模版
    * 对应的`pages`中也有与之对应的用法,请详细查考哦！
